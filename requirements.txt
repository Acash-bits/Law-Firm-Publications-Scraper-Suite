# Installation:
#   pip install -r requirements.txt
#
# For development (with optional packages):
#   pip install -r requirements-dev.txt
#
# To generate requirements from current environment:
#   pip freeze > requirements.txt
#
# To update all packages:
#   pip install --upgrade -r requirements.txt
#
# Virtual Environment Setup:
#   python -m venv venv
#   source venv/bin/activate  # Linux/Mac
#   venv\Scripts\activate     # Windows
#   pip install -r requirements.txt
#
# Python Version Compatibility:
#   - Tested with Python 3.8, 3.9, 3.10, 3.11
#   - Minimum required: Python 3.8+
#
# Platform-Specific Notes:
#
#   Windows:
#   - selenium may require Visual C++ Redistributable
#   - lxml may require additional build tools
#
#   macOS:
#   - May need to install Xcode Command Line Tools
#   - brew install python3
#
#   Linux:
#   - Ubuntu/Debian: sudo apt-get install python3-dev libxml2-dev libxslt1-dev
#   - CentOS/RHEL: sudo yum install python3-devel libxml2-devel libxslt-devel
#
# Selenium ChromeDriver:
#   - Option 1: Install via webdriver-manager (automatic)
#   - Option 2: Manual download from https://chromedriver.chromium.org/
#   
#   Linux: sudo apt-get install chromium-chromedriver
#   macOS: brew install chromedriver
#   Windows: Download and add to PATH
#
# MySQL Requirements:
#   - MySQL Server 5.7+ or 8.0+
#   - Create database: CREATE DATABASE publications_db;
#   - Create user with privileges (see README.md)
#
# Troubleshooting:
#
#   If installation fails for lxml:
#     pip install --upgrade pip setuptools wheel
#     pip install lxml
#
#   If mysql-connector-python fails:
#     Try: pip install mysql-connector-python --no-binary mysql-connector-python
#
#   If selenium ChromeDriver issues:
#     pip install webdriver-manager
#     Use ChromeDriverManager().install() in code
#
# Security Note:
#   Always review dependencies for security vulnerabilities
#   Use: pip-audit or safety check
